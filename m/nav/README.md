# nav

````html
<div class="m-nav">
    <span class="m-nav-item">html</span>
    <span class="m-nav-item">css</span>
    <!-- m-nav-item--on 是选中状态 -->
    <span class="m-nav-item m-nav-item--on">js</span>
</div>
````
